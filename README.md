FleaBug: GoLang Dumper
=========================

FleaBug provides an easy and pretty `fmt.Println` alternative to `Go Lang` projects. 

```go
fleabug.Dump("wopwop")
fleabug.Dump(1, 2, 3)
````

```go
wopwop  (string)

1 (int)
2 (int)
3 (int)
````
